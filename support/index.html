Use the routes below if you hit a bug, need purchase help, or want to verify how a platform service behaves in the current release.

Purchase help

  • Open the in-game Cosmetic Store.
  • Use the Restore button in the top-left corner.
  • If the item is still missing, confirm the same Apple ID is in use.
  • For billing issues, use Apple's purchase support channels.

Cloud save behaviour

Fleet Commander attempts CloudKit sync through the player's private iCloud account. If iCloud is unavailable or denied, the app falls back to local storage. The game should still launch and remain playable.

Game Center

Game Center sign-in is optional. Achievements and leaderboard submissions work only when the player is signed in. If Game Center is unavailable, the rest of the game should continue normally.

Before reporting a bug, note the app version shown in Settings and whether the issue occurred after restoring purchases, signing in to Game Center, or enabling iCloud sync.
