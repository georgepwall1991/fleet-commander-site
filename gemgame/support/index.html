Support

GemGame

Use the routes below if you hit a bug, need help restoring the Hard Mode Pack, or want to report a reproducible issue from the current release.

Fastest support route

Email support with your device model, iOS version, app version, build number, and the exact steps needed to reproduce the problem.

georgewall1991@icloud.com

Hard Mode Pack restore

  • Open the in-game shop from the menu or level map.
  • Use the restore button on the Hard Mode Pack card.
  • Confirm you are signed in with the same Apple ID used for the purchase.
  • For billing issues, use Apple's purchase support channels.

Useful bug report details

  • The level number or garden screen where the issue happened
  • Whether the problem occurred after a fresh install or after restoring purchases
  • Whether the issue happens every time or only occasionally
  • A screenshot or screen recording when possible

What to expect

GemGame should launch directly to the main menu, allow offline play, and keep local progress between launches. The Hard Mode Pack is a one-time unlock and should remain available after using the restore flow on the same Apple account.

Before reporting a bug, include the app version and build number shown in the app so the report can be matched to the shipped release.
