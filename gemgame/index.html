Cozy Match 3 Garden Puzzle Game

Clear boards. Earn stars. Restore the garden.

GemGame is a handcrafted match 3 puzzle game for iPhone. Complete level goals, collect stars, unlock neglected areas, and choose how restored spaces should look. The base route leads into an optional one-time Hard Mode Pack that unlocks 50 more levels after level 55.

What the game includes

  • Handcrafted match 3 puzzle levels with score, collect, and clear goals
  • Stars that feed directly into garden restoration progression
  • Decoration choices for restored areas
  • Coins, boosters, and a straightforward in-game shop
  • A one-time Hard Mode Pack that unlocks levels 56 to 105

Store model

GemGame is a paid-upfront iPhone game with a single optional non-consumable purchase. The Hard Mode Pack permanently unlocks 50 tougher levels on top of the main route.

The game contains no third-party ads and no recurring subscriptions.

Data and services

Progress and settings are stored on device. Purchase processing and restore flows use Apple's App Store infrastructure.
